*Cost Driver Analysis & Downtime Optimization in Semiconductor Manufacturing* 

-Project Overview-
This project analyzes maintenance data from semiconductor test equipment to identify key cost drivers and optimize maintenance decision-making using data analysis techniques.

-Objective-
The goal of this project is to:

*Identify the components generating the highest repair costs
*Analyze the relationship between failure frequency and cost impact
*Improve prioritization of maintenance strategies using data

-Dataset-
The dataset includes:

*Component name
*Number of failures
*Repair cost
*Cost avoided

Total annual maintenance cost analyzed: ~$500,000 USD

-Tools & Technologies-
*Python
*Pandas
*Matplotlib

-Analysis Performed-

1. Pareto Analysis

Identified that approximately 20% of components account for ~78% of total maintenance cost.

2. Failure vs Cost Analysis

Discovered that components with the highest failure frequency contribute to less than ~35% of total cost impact, highlighting inefficient prioritization.

3. Quadrant Analysis

Classified components based on:

Failure frequency
Cost impact

This enabled identification of low-frequency but high-cost critical components.

4. ROI Analysis

Calculated Return on Investment (ROI) for each component based on avoided cost vs repair cost.

This allowed prioritization of components with the highest financial impact and optimization potential.

-Key Insights-
*High failure frequency does not necessarily correlate with highest cost impact
*A small subset of components drives the majority of maintenance cost
*ROI analysis enables better prioritization of maintenance efforts
*Data-driven strategies improve operational efficiency

-Business Impact-
*Identified opportunity to reduce maintenance costs by approximately 15% (~$75,000 USD annually)
*Improved maintenance prioritization strategy
*Enabled better allocation of maintenance resources

-Conclusion-
This project demonstrates how data analysis can be applied in semiconductor manufacturing to optimize maintenance strategies, reduce costs, and improve operational performance.
